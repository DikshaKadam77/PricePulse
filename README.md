# PricePulse 

PricePulse is a modern expense tracker web application built using HTML, CSS, JavaScript, and Firebase.

## Features

- User Authentication (Firebase)
- Dashboard
- Add Expenses
- Transactions History
- Search Transactions
- Category Filter
- Delete Transactions
- Monthly Budget Tracking
- Responsive UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Firebase Authentication
- Cloud Firestore
- Font Awesome
